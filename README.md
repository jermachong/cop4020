# COP4020 - Programming Languages

This repository is used to store coursework and track progress for my Programming Languages course.

## Course Overview

For the first few weeks, the course focuses on the theoretical foundations of programming languages, including:

- Syntax and semantics of programming languages
- How language features are supported across different programming languages

After the theoretical portion, the course shifts to brief practical coverage of:

- C++
- Python
- Haskell
- JavaScript

## Professor's Course Description

"Theoretical basis for programming languages will be given for a few weeks. Semantics and syntax of programming languages, and how they are supported by different programming languages, will be described. After the theoretical view, C++, Python, Haskell and Javascript will be covered briefly until the end of semester."

- Keep course materials organized

- Maintain a semester-long record of learning and implementation work
